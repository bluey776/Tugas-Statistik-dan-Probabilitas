# Analisis Regresi Linear Berganda: Hubungan Screen Time dan Durasi Tidur terhadap Tingkat Kecemasan

Repository ini dibuat untuk memenuhi tugas mata kuliah Statistika dan Probabilitas. Proyek ini bertujuan untuk memprediksi tingkat kecemasan berdasarkan pola perilaku digital (screen time) dan pola istirahat (durasi tidur).

## Struktur Repository
* ScreenTime vs MentalWellnes.csv : Dataset sekunder berskala besar (N >= 100) yang digunakan sebagai acuan parameter awal.
* dataset_survey.csv : Dataset primer skala kecil hasil survei mandiri terhadap mahasiswa lokal/lingkup sekitar.
* regresi_anxiety.py : Script Python untuk melakukan perhitungan regresi

## Variabel yang Digunakan
* Rata-rata Screen Time Harian (Jam)
* Rata-rata Durasi Tidur Malam (Jam)
* Skala Tingkat Kecemasan (1-10)
